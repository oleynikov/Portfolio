Форма регистрации

Функциональная, мощная, стильная, кроссбраузерная user-friendly форма регистраци на XHTML, CSS3, Java Script, PHP.

Используюется передовая технологи AJAX асинхронного обмена данными с сервером для проверки доступности для регистрации указанных логина и электронной почты без перезагрузки страницы.

Не использовано ни одной сторонней библиотеки. Применяется авторская библиотека для создания анимации на Java Script + CSS.

Аудио плеер

Плеер представляет собой приложение для установки на веб-сервер. PHP скрипт сканирует директорию с музыкальными файлами и формирует XML документ с иерархической структурой музыкального каталога.

Клиентская часть приложения, написанная на связке Flash + Action Script 3, использует созданный ранее XML плейлист в файловом браузере для отображения доступных для воспроизведения композиций.

При написании приложения не использовано ни одной сторонней библиотеки.

Игра "Тетрис"

Простейший ремейк всемирно известной игры Алексея Пажитного на чистом Action Script 3.0

Сторонние библиотеки не использовались.

Управление: ←, →, ↑, ↓, пробел.

Школьное расписание

Программа предназначена для школ, имеющих в своем распоряжении плазменные или ЖК экраны больших размеров.

Серверная часть приложения написана на PHP и запрашивает расписание на следующий урок из базы данных и формирует соответствующий XML документ.

Клиентская часть программы, написанная на Action Script 3, загружает расписание в формате XML и отображает его в виде прокручивающейся таблицы на экране.

Следует особо отметить, что приложение поддерживает функцию гибкой настройки через дружелюбный пользовательский веб-интерфейс с последующим сохранением настроек в конфигурационный файл на сервере.

CMS "Коллекционер"

Система управления контентом "Коллекционер" является узкоспециализированной системой, предназначенной для использования коллекционерами любого профиля.

CMS предоставляет возможность создания и редактирования коллекций, а также предметов, составляющими их.

Find-Alike.Ru

Авторский сервис для поиска похожих по звучанию музыкальных исполнителей и коллективов.

Поддерживает возможность установки связей между музыкальными коллективами, а также поиск похожих исполнителей. Используются передовые технологии поиска.
